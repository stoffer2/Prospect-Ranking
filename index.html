Welcome to Rankle

"Like Rotten Tomatoes, but for baseball prospects."

📊 Rankle Score (0–100) — Our consensus metric, built by a model that weighs each prospect's placement across multiple expert rankings — not a simple average, but a calculated score that reflects both where and how consistently a prospect appears across lists.
🎯 Consensus dots — Lit dots show how many sources place a prospect in roughly the same tier. More dots = more agreement on where they belong.
⚡ Volatility — How much the experts disagree. Low = everyone's on the same page. High = polarizing prospect worth watching closely.
⚾ Fantasy roster upload — Import your league roster to instantly highlight which top prospects are still available.
